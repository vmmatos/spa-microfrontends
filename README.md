# Working with microfrontends

Building Micro Frontends with React, Vue, and Single-spa

## Instalation
Needs Node >= 10
```bash
npm install
```

## Running in dev mode
```bash
npm start
```

Open the browser, go to...
```bash
http://localhost:8080/
```

Renders only React
```bash
http://localhost:8080/react
```

Renders only Vue
```bash
http://localhost:8080/vue
```
